# Agencia_viaje_webII

Próximo destino

Este proyecto es una aplicación web para una agencia de viajes. Permite buscar y reservar vuelos y hoteles.

## Funcionalidades

- Búsqueda de vuelos
- Reserva de hoteles

## Cómo contribuir

- Crea una rama para cada nueva funcionalidad.
- Envía una pull request para revisión.

  ## Paquetes turisticos
- Sitios y viajes
